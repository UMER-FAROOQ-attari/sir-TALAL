**Map**
A JavaScript Map is an object that can store collections of key-value pairs, similar to a dictionary in other programming languages. (W3school)
**difference between Map and Object**
Maps differ from standard objects in that keys can be of any data type.
**Set**
A JavaScript Set is a collection of unique values.
Each value can only occur once in a Set.
The values can be of any type, primitive values or objects.
